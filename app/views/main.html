Google Maps for Angular.js

Simple to use directives to work with Google Maps inside your Angular.js applications.

Download View on Github

Features

  • Asynchronous loading of Google Maps Javascript API v3 with configurable API key and sensor flag;
  • Directives for maps, markers, info windows, polylines and more;
  • Services for traffic, directions and geocoding

Use it

$ bower install angular-google-maps --save
and read the Getting started page.

Developer friendly

angular-google-maps uses Grunt as its main build system and Jasmine as its testing framework.

Announcements

Working on polyline integration
We're currently working to the integration of polylines. This will introduce a new directive, polyline, which will be used inside the google-map directive to draw a line between a list of coordinates.
